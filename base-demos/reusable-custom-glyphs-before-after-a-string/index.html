Reusable Glyphs

Implement custom glyphs using data-attribute in html and CSS.

I recommend using Icomoon.io for generating a custom web font.

Classes

.u-glyph-before[data-icon]

Add a custom glyph before text, change the [data-icon] to your desired character or the unicode. 

<span class="u-glyph-before" data-icon="X">
	I have an "X" before me
</span>

I have an "X" before me

.u-glyph-after[data-icon]

Add a custom glyph after text, change the [data-icon] to your desired character or the unicode. 

<span class="u-glyph-after" data-icon="X">
	I have an "X" after me
</span>

I have an "X" after me

.u-glyph

Replace a character or unicode with a glyph 

<span class="u-glyph">
	X
</span>

X

.u-glyph--no-speak

Prevent screen readers from reading your character. 

<span class="u-glyph--no-speak">
	X
</span>

X
